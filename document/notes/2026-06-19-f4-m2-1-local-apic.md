# F4-M2 批1:Local APIC(xAPIC MMIO)驱动

> 2026-06-19。F4-M2 批 1。范围:LocalAPIC driver(xAPIC MMIO)+ mock 单测。不切换(M2-3)/不接 main。
> 基线 = M1-4(859/0)。

## 决策
- **xAPIC MMIO 必选**:qemu64 无 x2APIC(M1 调研定);x2APIC(MSR)留可选。
- **MMIO 单独 map + FLAG_PCD**:APIC 寄存器是 memory-mapped I/O 非 RAM,**严禁经 direct-map cache-enabled 访问**(未定义行为)。init() 在 `KMEM_MMIO_BASE + 0x10000`(避开 AHCI BAR5 占的 KMEM_MMIO_BASE)map 4KB + `FLAG_PRESENT|WRITABLE|PCD`。
- **bind()/init() 分离**:init() map 真 MMIO(production);bind() 注入已映射/mock base(测试)。mock test 用 RAM 页验证 driver 逻辑,不碰真硬件(降 M2-1 风险,切换留 M2-3)。
- **M2-1 不接线**:driver + g_lapic 全局 + mock test only;production init/enable 在 M2-3 切换调。

## 实现
- `local_apic.hpp`:LocalAPIC(bind/init/read/write/id/version/enable/disable/eoi/error_clear)+ 寄存器偏移常量 + kSvrEnable + g_lapic extern。
- `local_apic.cpp`:实现(init map+PCD;read/write 经 volatile 指针 + 字节偏移;id 取 bits24-31;enable SVR vector|bit8;eoi 写 0;clear_error 写 0)。
- `test_apic.cpp`:6 mock 测(read/write roundtrip / enable SVR / disable / eoi=0 / id 解码 / clear_error),MockMmio(volatile uint32_t[256] RAM 页)。

## 验证
- `timeout 40 run-kernel-test`:**865/0**(859 + 6 mock)。零编译警告。
